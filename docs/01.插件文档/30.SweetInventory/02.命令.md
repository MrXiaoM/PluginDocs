根命令为 `/sweetinventory`，别名为 `/sinv`、`/si` 或 `/menu`。  
使用 `<>` 包裹的为必选参数，使用 `[]` 包裹的为可选参数。

| 命令 | 说明 | 权限 |
| --- | --- | --- |
| `/menu open <菜单>` | 打开指定菜单，需要打开菜单所需权限 | `sweet.inventory.open` |
| `/menu open <菜单> - [..菜单参数]` | 打开指定菜单，需要打开菜单所需权限 | `sweet.inventory.open` |
| `/menu open <菜单> <玩家> [..菜单参数]` | 为玩家打开指定菜单，无需指定玩家拥有打开菜单所需权限 | `sweet.inventory.open.other` |
| `/menu list` | 查看菜单列表 | `sweet.inventory.list` |
| `/menu reload` | 重载插件配置文件 | OP/控制台 |

打开菜单需要 `sweet.inventory.open.menu.<菜单>` 权限。
