## 经济接口

本插件允许其它插件向其中注册自定义货币类型。

```java
SweetAdaptiveShop plugin = SweetAdaptiveShop.getInstance();
plugin.registerEconomy(resolver); // 向插件注册货币解析器
```

自带的货币类型兼容详见包 [`top.mrxiaom.sweet.adaptiveshop.api.economy`]() 中的实现类，用户使用格式如下：
+ `Vault` 代表 Vault 金币
+ `PlayerPoints` 代表 PlayerPoints 点券
+ `MPoints:sign` 代表 MPoints 插件的 sign 点数
+ `CoinsEngine:currency` 代表 CoinsEngine 的 currency 货币
