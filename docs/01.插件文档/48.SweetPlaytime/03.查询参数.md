> 查询参数的解析逻辑详见 [`Query#parse`](https://github.com/MrXiaoM/SweetPlaytime/blob/main/src/main/java/top/mrxiaom/sweet/playtime/config/Query.java)

查询参数用于获取玩家的累计在线时间，一次查询可以添加多个查询参数，用于指定查询的日期范围、标签等。当前可用的查询参数如下：

+ `today` - 将日期范围设置在今天内
+ `current_month`|`this_month`|`current month`|`this month` - 将日期范围设置在这个月内
+ `current_year`|`this_year`|`current year`|`this year` - 将日期范围设置在今年内
+ `Monday`|`Tuesday`|`Wednesday`|`Thursday`|`Friday`|`Saturday`|`Sunday` - 将起始日期设置在本周第几天，并将结束日期设置为起始日期的7天后
+ `tag=标签` - 设置查询标签
+ `start=yyyy-MM-dd` - 设置固定起始日期
+ `end=yyyy-MM-dd` - 设置固定结束日期

如无特殊说明，均可忽略大小写。
