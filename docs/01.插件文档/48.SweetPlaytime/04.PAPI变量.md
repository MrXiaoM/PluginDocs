```
%sweetplaytime_query% 查询玩家已被记录的总在线时间秒数
%sweetplaytime_query_<查询参数>% 查询玩家的在线时间秒数
%sweetplaytime_format_query% 查询玩家已被记录的总在线时间，格式化为 X天X时X分X秒
%sweetplaytime_format_query_<查询参数>% 查询玩家的在线时间，格式化为 X天X时X分X秒
%sweetplaytime_status_<奖励配置>:all% 获取某个奖励配置中，任意奖励的领取状态
%sweetplaytime_status_<奖励配置>:<时间>% 获取某个奖励配置中，单独奖励的领取状态
```

查询参数详见[文档说明](/docs/playtime/query)，多个查询参数使用英文逗号`,`隔开。示例如下
```
%sweetplaytime_query_today% 查询玩家今天的在线时间秒数
%sweetplaytime_query_today,tag=foo% 查询玩家今天在 foo 标签下的在线时间秒数
```

status 变量返回的领取状态如下
```
0 - 没有可领取的奖励
1 - 可领取奖励
2 - （单独奖励）已经领取了
```
