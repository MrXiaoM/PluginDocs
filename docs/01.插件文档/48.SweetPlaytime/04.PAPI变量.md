```
%sweetplaytime_query% 查询玩家已被记录的总在线时间秒数
%sweetplaytime_query_<查询参数>% 查询玩家的在线时间秒数
```

查询参数详见[文档说明](/docs/playtime/query)，多个查询参数使用英文逗号`,`隔开。示例如下
```
%sweetplaytime_query_today% 查询玩家今天的在线时间秒数
%sweetplaytime_query_today,tag=foo% 查询玩家今天在 foo 标签下的在线时间秒数
```
