根命令为 `/sweetplaytime`，别名 `/playtime`、`/spt`、`/pt`。  
其中 `<>` 包裹的为必选参数，`[]` 包裹的为可选参数。

| 命令 | 说明 | 权限 |
| --- | --- | --- |
| `/pt cleanup` | 立即异步执行数据库清理操作 | OP |
| `/pt reload database` | 重载并重新连接到数据库 | OP |
| `/pt reload` | 重载插件配置文件 | OP |
