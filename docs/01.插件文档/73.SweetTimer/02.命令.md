根命令为 `/sweettimer`，别名为 `/stimer`, `/timer`  
以 `<>` 包裹的为必选参数，以 `[]` 包裹的为可选参数。

玩家命令
| 命令 | 描述 | 权限 |
| --- | --- | --- |
| `/timer print <定时器>` | 获取定时器配置详情 | OP/控制台 |
| `/timer condition <定时器>` | 测试定时器设定条件是否满足 | OP/控制台 |
| `/timer reload` | 重载插件配置文件 | OP/控制台 |
