## 命令

根命令 `/sweetchat`，别名为 `/chat`、`/schat` 或 `/sc`。  
`<>`包裹的为必选参数，`[]`包裹的为可选参数。  

| 命令 | 描述 | 权限 |
| --- | --- | --- |
| `/chat sudo <玩家> <消息内容>` | 强制某个在线玩家发送聊天消息 | OP/控制台 |
| `/chat reload database` | 重新连接数据库 | OP/控制台 |
| `/chat reload` | 重载配置文件 | OP/控制台 |

## 权限

+ `sweetchat.style.键名` - 允许使用 `styles.yml` 中的样式
+ `sweetchat.local.bypass` - 是否允许看到当前子服的所有本地聊天（附近聊天），即使说话人不在玩家附近

## 消息标签权限

指定玩家可以在消息中使用哪些 MiniMessage 标签。

+ `sweetchat.format.legacy` - 有此权限时，将旧版格式 (`&`) 转换为 MiniMessage
+ `sweetchat.format.color.all` - 允许在消息中使用所有[颜色代码](https://zh.minecraft.wiki/w/%E6%A0%BC%E5%BC%8F%E5%8C%96%E4%BB%A3%E7%A0%81?variant=zh-cn#%E9%A2%9C%E8%89%B2%E4%BB%A3%E7%A0%81)
  - `sweetchat.format.color.hex` - 允许在消息中使用16进制颜色 (如 `<#FF06B5>`)
  - `sweetchat.format.color.named` - 允许在消息中使用已命名颜色
    - `sweetchat.format.color.black` - 允许在消息中使用黑色颜色 (`<black>`、`&0`)
    - `sweetchat.format.color.dark-blue` - 允许在消息中使用蓝色颜色 (`<dark_blue>`、`&1`)
    - `sweetchat.format.color.dark-green` - 允许在消息中使用绿色颜色 (`<dark_green>`、`&2`)
    - `sweetchat.format.color.dark-aqua` - 允许在消息中使用青色颜色 (`<dark_aqua>`、`&3`)
    - `sweetchat.format.color.dark-red` - 允许在消息中使用红色颜色 (`<dark_red>`、`&4`)
    - `sweetchat.format.color.dark-purple` - 允许在消息中使用品红颜色 (`<dark_purple>`、`&5`)
    - `sweetchat.format.color.gold` - 允许在消息中使用黄色颜色 (`<gold>`、`&6`)
    - `sweetchat.format.color.gray` - 允许在消息中使用淡灰色颜色 (`<gray>`、`&7`)
    - `sweetchat.format.color.dark-gray` - 允许在消息中使用灰色颜色 (`<dark_gray>`、`&8`)
    - `sweetchat.format.color.blue` - 允许在消息中使用亮蓝色颜色 (`<blue>`、`&9`)
    - `sweetchat.format.color.green` - 允许在消息中使用亮绿色颜色 (`<green>`、`&a`)
    - `sweetchat.format.color.aqua` - 允许在消息中使用亮青色颜色 (`<aqua>`、`&b`)
    - `sweetchat.format.color.red` - 允许在消息中使用亮红色颜色 (`<red>`、`&c`)
    - `sweetchat.format.color.light-purple` - 允许在消息中使用亮品红颜色 (`<light_purple>`、`&d`)
    - `sweetchat.format.color.yellow` - 允许在消息中使用亮黄色颜色 (`<yellow>`、`&e`)
    - `sweetchat.format.color.white` - 允许在消息中使用白色颜色 (`<white>`、`&f`)
+ `sweetchat.format.color.shadow` - 允许在消息中设置阴影颜色 (如 `<shadow:#00000000>`)
+ `sweetchat.format.gradient` - 允许在消息中使用渐变色格式 (`<gradient>`)
+ `sweetchat.format.decoration.all` - 允许在消息中使用所有[格式代码](https://zh.minecraft.wiki/w/%E6%A0%BC%E5%BC%8F%E5%8C%96%E4%BB%A3%E7%A0%81?variant=zh-cn#%E6%A0%BC%E5%BC%8F%E4%BB%A3%E7%A0%81)
  - `sweetchat.format.decoration.bold` - 允许在消息中使用加粗格式 (`<bold>`、`<b>`、`&b`)
  - `sweetchat.format.decoration.italic` - 允许在消息中使用斜体格式 (`<italic>`、`<em>`、`<i>`、`&o`)
  - `sweetchat.format.decoration.magic` - 允许在消息中使用随机乱码格式 (`<obfuscated>`、`<obf>`、`&k`)
  - `sweetchat.format.decoration.delete` - 允许在消息中使用删除线格式 (`<strikethrough>`、`<st>`、`&m`)
  - `sweetchat.format.decoration.underline` - 允许在消息中使用下划线格式 (`<underlined>`、`<u>`、`&n`)
+ `sweetchat.format.hover` - 允许在消息中使用鼠标悬停显示格式 (`<hover>`)
+ `sweetchat.format.click` - 允许在消息中使用点击操作格式 (`<click>`)
+ `sweetchat.format.insertion` - 允许在消息中使用插入内容格式 (`<insertion>`)
+ `sweetchat.format.translatable` - 允许在消息中使用可翻译文本格式 (`<lang>`)
+ `sweetchat.format.font` - 允许在消息中使用字体格式 (`<font>`)
