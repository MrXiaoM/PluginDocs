命令与权限

根命令为 /sweetdrops，别名 /sdrops, /sd。
<>包裹的是必选参数，[]包裹的是可选参数，
所有命令均只能由管理员或控制台执行

命令 描述
/sd debug 开启或关闭调试模式
/sd prefeb <预制体> [数量] [玩家] 给予自己或某人指定数量的预制体
/sd reload 重载配置文件
